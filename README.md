# CCAF WhatsApp Share (WhatsApp Cards Builder)

An interactive, premium web utility designed to help candidates prepare for and share knowledge about the **CCAF Exam**. This tool acts as an active study aid that structures and formats complex conceptual notes into clean, copy-pasteable WhatsApp markdown messages.

---

## 🎯 Purpose of the Site

The CCAF exam tests deep conceptual knowledge across multiple technical domains:
* **D1**: Agentic Architecture
* **D2**: Tool Design & MCP
* **D3**: Claude Code Config
* **D4**: Prompt Engineering
* **D5**: Context Management

When studying, candidates often encounter subtle traps, register sudden conceptual clicks, or discover helpful references. **CCAF WhatsApp Share** provides a structured way to document these insights instantly and format them specifically for sharing on WhatsApp study groups, peers, or self-reference threads.

---

## 💡 Why it is Useful

1. **Automated WhatsApp Markdown**: Typing custom markdown formatting (e.g., `*bold*`, `_italics_`, emojis, line breaks, and list indents) manually on a phone keyboard is tedious and error-prone. This app formats it dynamically in a live WhatsApp-styled preview.
2. **Encouraged Best Practices (Active Recall & Feynman Technique)**: The builder enforces structure based on the type of note you are making:
   - **Traps Caught**: Forces you to contrast your *Wrong Assumption* with *What is Actually True*, highlighting the key distinction so you don't repeat the mistake.
   - **Concepts Clicked**: Prompting for *Analogies* and *Technical Realities* to explain complex mechanisms in plain language.
   - **Resources/Notes**: Organizes pages, sections, and relevance for quick bookmark sharing.

---

## 📚 How it Helps You Study & Share

* **Active Recall**: The form fields guide you to think about the core explanation (e.g., "What actually happens", "How the exam tests this trap"), rather than just reading passively.
* **Peer Learning**: Explaining concepts to others is one of the most effective ways to solidify your own understanding. The **"Copy for WhatsApp"** button makes it a single-click action to share premium-formatted insights with your peer study groups.
* **Quick Reference Log**: Use the categorized cards to keep a structured record of all 10 weeks of material.

## 🛠️ Getting Started & Installation

To run this application locally, ensure you have Node.js installed, then execute:

```bash
# 1. Install dependencies
npm install

# 2. Run the Vite development server
npm run dev

# 3. Compile a production build
npm run build
```
